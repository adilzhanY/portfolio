# Portfolio — Adilzhan Yerzhan (`qantrr`)

Personal portfolio site for **Adilzhan Yerzhan**, a Software Engineering student & full-stack developer based in Potsdam, Germany.

Built with [Astro](https://astro.build), React islands, Tailwind CSS, and GSAP animations — featuring a custom cursor, an animated marquee, and a WebGL-style black hole background.

## 🛠️ Tech Stack

- **[Astro](https://astro.build)** — content-driven static site framework
- **[React](https://react.dev)** — interactive islands (cursor, loader)
- **[Tailwind CSS](https://tailwindcss.com)** — styling
- **[GSAP](https://gsap.com)** — animations
- **TypeScript** — CV data & type-safe content

## 🚀 Getting Started

```sh
npm install       # install dependencies
npm run dev       # start dev server at localhost:4321
```

## 🧞 Commands

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Installs dependencies                        |
| `npm run dev`     | Starts local dev server at `localhost:4321`  |
| `npm run build`   | Builds the production site to `./dist/`      |
| `npm run preview` | Previews the production build locally        |
